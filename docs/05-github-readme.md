# GitHub・README仕様

## GitHub保存

- 新規ファイル・変更ファイルをGitHubに反映する
- 分かりやすいコミットメッセージでコミットする
- Firebaseの設定値やAPIキーは `.env` に分離する
- `.env` は `.gitignore` に追加し、GitHubにはアップロードしない
- `.env.example` を作成する
- README.md にセットアップ方法、起動方法、Firebase設定方法を書く
