# 機能仕様

## 必須機能

- 企業追加、編集、削除
- 企業検索、業界別フィルター、選考状況フィルター、志望度フィルター、難易度フィルター、タグ検索
- 本命、インターン、締切間近などのカテゴリ表示
- 締切、志望度、難易度、優先度、最近更新順の並び替え
- メモ、タグ、URL、締切、選考状況管理
- 今日やること、次に対応すべき企業、締切間近の強調表示
- 空状態、ローディング、エラー表示

## 企業登録・編集フォーム

登録と編集は同じフォームで行い、必須項目は企業名だけにする。

入力項目は、企業名、業界、職種、志望度、難易度、選考状況、マイページ登録状況、ES提出状況、SPI・適性検査、面接予定、インターン応募状況、応募予定日、ES締切、インターン締切、次の締切日、次にやること、各種メモ、URL、タグ。

## 表示切り替え

- カード表示
- コンパクト表示
- 一覧表表示
- 選考状況ごとのカンバン表示

## 自動判定

- 締切3日以内は「至急」
- 締切7日以内は「注意」
- 締切超過は「期限切れ」
- 次にやることが未入力なら「次アクション未設定」
- 長期間更新されていない企業は「放置中」
- 志望度、締切、選考状況、次アクションから優先度を計算する
