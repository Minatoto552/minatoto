# 就活Webアプリ概要

## 目的

スマホ・PCの両方で使える「就活専用Webアプリケーション」を作成する。

就活中の学生が、応募企業、インターン、選考状況、締切、次にやること、メモ、志望度、難易度、マイページURL、採用ページURLを一元管理できるようにする。

## 最重要方針

- ログイン画面は作らない
- 最初から企業一覧・ダッシュボード画面を表示する
- 既存企業データ・サンプル企業は入れない
- 企業はユーザーが手動で登録する
- 企業登録画面と企業編集画面は1つに統一する
- ダウンロード機能は不要
- Firebase Firestore に企業データを保存する
- スマホ・PCの両方に対応する
- PCでは一画面にできるだけ多くの企業が見えるようにする
- スマホでは片手で操作しやすいレイアウトにする

## 技術構成

- React
- TypeScript
- Vite
- Firebase Firestore
- Tailwind CSS または CSS Modules
