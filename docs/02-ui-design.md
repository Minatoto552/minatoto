# UI・デザイン仕様

## 方向性

白・薄いグレーを基調に、清潔感あるカードUIと就活サイトの一覧性を組み合わせる。ロゴ、文言、画像、商標は使用しない。

## 画面構成

- ヘッダー: アプリ名「就活企業管理」、検索、並び替え、フィルター、表示切り替え、企業追加
- カテゴリナビゲーション: すべて、本命、インターン、選考中、マイページ作成済み、ES提出前、SPI・適性検査、面接予定、締切間近、内定候補、見送り
- ダッシュボード: 登録企業数、選考中、締切間近、本命、今日やること、明日まで、3日以内、優先度高、放置中、次に対応すべき企業
- 企業一覧: PCは4〜6列程度、スマホは1列

## レスポンシブ

- PC: 一覧性を重視し、カードを多列表示する
- スマホ: 検索バーを上部に固定し、カテゴリは横スクロール、追加ボタンは右下固定にする
