# AGENTS.md

## 作業ルール

このリポジトリでは、就活専用Webアプリケーションを作成・保守します。

- 日本語で作業内容を説明する
- 既存コードがある場合は、壊さずに必要な変更だけを行う
- 作業前に現在のプロジェクト構成を確認する
- 実装前に変更方針を簡単に整理する
- React + TypeScript + Vite を前提にする
- Firebase Firestore を使用する
- Firebase の設定値は `.env` に分離する
- `.env` は GitHub にアップロードしない
- `.env.example` を作成する
- README.md にセットアップ方法を書く
- スマホ・PCの両方で使いやすいUIにする
- ログイン画面は作らない
- サンプル企業データは表示しない
- ダウンロード機能は不要
- 作業後、変更ファイルと実装内容をまとめる
